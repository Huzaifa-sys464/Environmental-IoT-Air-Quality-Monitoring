# Environmental IoT Air Quality Monitoring System

An ESP32-based environmental monitoring system that measures dust particles, gas concentration, and temperature and triggers alerts when pollution levels exceed safe thresholds.

## Overview

Air pollution monitoring is critical in industrial environments.  
This project demonstrates a low-cost IoT monitoring system capable of detecting environmental changes in real time.

The system uses ESP32 with environmental sensors to measure:

- Dust particles
- Gas concentration
- Temperature

When values exceed predefined thresholds, the system triggers alerts.
